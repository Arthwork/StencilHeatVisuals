# 2D Heat Transfer (Stencil) — project root

- **`code/`** — C sources, `Makefile`, `compare_outputs.py`, experiment and Slurm scripts, and the technical README for build and run commands. **Do all `make` and test runs from `code/`.**
- **`data/`** — Collected run outputs, timing tables, and **`README.md`** (master plan: experiments, QA, and figures). Default experiment output for `run_experiments.sh` is under `../data/results` when run from `code/`.
- **`presentation/`** — Placeholder for presentation PDF and related assets / links.

For instructions and the full project checklist, start with `data/README.md`.
